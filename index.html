Deployment recommendation

Use GitHub Pages for this portfolio.

This site is static HTML and CSS, so GitHub Pages is the better fit: it is free, simple, reliable, and gives the portfolio a clean public URL. Railway is excellent for apps with servers, databases, background workers, environment variables, or persistent volumes. For this portfolio page, Railway would add cost and operational surface without adding much value.
